### PHI-REL-003: Chat Service notifyOwner Goroutine Without Panic Recovery

**Location**: `internal/service/ai/chat_service.go:301`

**Problem**: Fire-and-forget goroutine without panic recovery:

```go
func (cs *ChatService) notifyOwner(_ context.Context, html string) {
    if cs.ownerNotify == nil {
        return
    }
    go cs.ownerNotify(context.Background(), html)  // ← No recover
```

**Risk**: If `ownerNotify` callback panics, the entire bot crashes.

**Acceptance Criteria**:
- Add `defer recover()` inside the goroutine
- Log recovered panic
- Maintain fire-and-forget behavior

**Priority**: P2 (Medium)  
**Effort**: XS (~10 minutes)
